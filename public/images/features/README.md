# Feature Screenshots

Place your feature screenshots in this directory with the following names:

1. **tests.png** - Screenshot of the Tests feature
2. **canvas.png** - Screenshot of the Canvas feature  
3. **quick-questions.png** - Screenshot of the Quick Questions feature

## Image Guidelines

- Format: PNG, JPG, or WebP
- Recommended size: 1200x800px or similar 3:2 aspect ratio
- Keep file size under 500KB for optimal loading

The images will be displayed in the waitlist page feature cards with a 16:9 aspect ratio (h-64).
